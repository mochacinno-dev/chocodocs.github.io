Installation¶

This guide will help you install Choco on your system.

Right now, the only way of installing it is by cloning it in your system, you can either clone the GitHub repository using Git by doing:

git clone https://github.com/mochacinno-dev/chocolang.git

Or downloading it directly by:

  1. Visiting the repository
  2. Click the green "Code" button
  3. Select "Download ZIP"
  4. Extract the ZIP file to your desired location
  5. Navigate to the extracted folder.

Running Tourmaline¶

Interactive Mode (REPL)¶

Start the CocoaInterpreter:

choco

NOTE: This only works if choco.exe has been added to PATH

Running Script Files¶

Execute a Choco script file:

choco <file>.choco

Tip

Choco only supports the .choco file extension.
