Welcome to Choco

Choco is a high-level programming language, primarily intended for general use. It is designed to be a simple and convenient programming language for developers, borrowing syntax elements from Ruby, Go, and Rust!

fn greet(name) {
    return "Hello, #{name}!";
}

let greeting = greet("World");
puts greeting;

Why Choose Choco?

  • Easy to use: Simple, beautiful syntax that doesn't sacrifice readability or functionality

  • Expressive: Write what you mean in a natural, readable way.

  • Duck-based: If it looks like a duck and walks like a duck, then it must be a duck.

  • Cross-Platform: Runs anywhere C/C++ runs.
